# Summary
This is a small JRPG project that I've been working on. The purpose of the project was to learna bout and implement JPRG battle systems, with the game state reflecting and being controlled by player menus

A "challenge mode" exists which puts the player in preset battles to test their knowledge of the game systems and to show off some of interesting interactions between character abilities
 
This project was made in [Godot](https://github.com/godotengine/godot) v4.4.1 with GDScript

# Controls
- Movement: WASD / Xbox LStick
- Strafe: LShift / Xbox RB
- Interact: E / Xbox A
- Open Menu: Q / Xbox Y
- Back: Esc / Xbox B
- Quit Dungeon: Ctrl+Shift+Z
