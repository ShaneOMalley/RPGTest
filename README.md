# Summary
This is a small JRPG project that I've been working on. The purpose of the project was to learna bout and implement JPRG battle systems, with the game state reflecting and being controlled by player menus

The main game has some interesting possible interactions between different characters' class abilities, but because the focus of the personal project was to implement JRPG systems, game balance around the difficulty and pacing of enemy encounters was not a priority. A "challenge mode" exists which puts the player in preset battles to test their knowledge of the game systems and to show off some of these interesting interactions. There is one solution per challenge, and they are designed to force the player to use strategy and their knowledge of character abilities and game systems to complete.
 
This project was made in [Godot](https://github.com/godotengine/godot) v4.4.1 with GDScript

# Controls
- Movement: WASD / Xbox LStick
- Strafe: LShift / Xbox RB
- Interact: E / Xbox A
- Open Menu: Q / Xbox Y
- Back: Esc / Xbox B
- Quit Dungeon: Ctrl+Shift+Z
